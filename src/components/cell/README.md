## Cell

列表项组件。

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | '' | 标题 |
| desc | string | '' | 描述 |
| arrow | boolean | false | 是否显示右侧箭头 |
| bordered | boolean | true | 是否显示底部分割线 |

### Slots

| 名称 | 说明 |
|------|------|
| icon | 左侧图标区域 |
| extra | 右侧额外内容（箭头之前） |

### Events

| 事件 | 说明 |
|------|------|
| tap | 点击时触发 |
