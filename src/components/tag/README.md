## Tag

彩色标签组件。

### Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| text | string | '' | 标签文本 |
| type | string | 'default' | 样式变体：default / primary / success / warning / danger |
| size | string | 'md' | 尺寸：sm / md |
| closable | boolean | false | 是否可关闭 |

### Events

| 事件 | 说明 |
|------|------|
| close | 点击关闭按钮时触发 |
